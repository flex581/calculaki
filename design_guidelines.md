# Calculaki - Design Guidelines

## Design Approach
**Reference-Based:** Clean, minimalist utility tools aesthetic inspired by modern calculator/converter sites with focus on speed and usability.

## Core Visual Identity

**Color Palette:**
- Base: Blue and white
- Clean, minimalist aesthetic
- Illustrative icons throughout

**Typography:**
- Modern, readable sans-serif font family
- Clear hierarchy: Large headings for tool names, medium for instructions, small for explanations
- Numbers in results: Extra large, bold display

## Layout System

**Spacing:** Consistent use of Tailwind spacing units: 2, 4, 6, 8, 12, 16 for padding/margins

**Page Structure:**
- Fixed header with logo "Calculaki" and navigation menu
- Full-width content sections
- Responsive grid layouts for category cards
